# 家庭健康管理中台 - 用户使用说明

## 1. 登录与首次初始化
1. 打开前端页面。
2. 若是首次部署，点击 `首次部署？去初始化 Owner`，填写用户名/密码/显示名，点击初始化。
3. 回到登录页，使用同一账号登录。

## 2. 设置中心（Owner/Admin）
1. 在左侧进入 `设置中心`。
2. 填写 Provider 名称、Base URL、API Key 后点击 `保存 Provider`。
3. 点击某个 Provider 的 `刷新模型`，更新模型目录。
4. 在 Runtime Profile 区填写模型 ID 与参数 JSON，点击创建。
5. 在 MCP 区域添加 MCP Server，并在多选框绑定到 `QA` Agent。

## 3. 聊天中心
1. 点击 `新建` 创建会话。
2. 右栏可多选 MCP，点击 `保存为会话默认 MCP`。
3. 可上传附件，系统会先脱敏再入库。
4. 输入问题后点击 `发送到 Agent`。

## 4. 知识库中心
1. 创建知识库。
2. 填写文档标题和内容后点击 `构建`。
3. 在右侧查看文档状态与构建结果。
4. 输入查询词点击 `检索` 查看命中片段。

## 5. 导出中心
1. 选择导出类型（聊天记录、知识库文档）。
2. 设置是否包含原始文件与脱敏文本。
3. 点击 `创建打包任务`。
4. 在任务列表可下载 ZIP 或删除任务。

## 6. 常见问题
- 登录失败：确认后端已启动，账号是否存在。
- 附件上传失败：可能命中未脱敏敏感信息，请先在设置中心增加脱敏规则。
- 导出下载失败：任务必须先进入 `done` 状态。
